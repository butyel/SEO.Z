# Suposições

1. O repositório estava vazio no início do desenvolvimento.
2. A aplicação será usada exclusivamente pelo proprietário.
3. O deploy será na Vercel com Supabase como backend.
4. O Google OAuth será configurado posteriormente pelo proprietário.
5. A chave de criptografia de tokens será gerada pelo proprietário.
6. O modelo de IA será definido pelo proprietário nas variáveis de ambiente.
7. O fuso horário padrão é America/Sao_Paulo.
8. A interface será integralmente em português do Brasil.
9. Não haverá migração de dados de sistemas externos.
10. O seed de demonstração usará dados fictícios claramente identificados.
